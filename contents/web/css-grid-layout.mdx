# CSS Grid Layout 완벽 가이드

CSS Grid Layout은 웹 페이지를 위한 2차원 레이아웃 시스템을 제공합니다. 행과 열을 모두 제어할 수 있어 복잡한 레이아웃을 간단하게 구현할 수 있습니다.

## Grid 기본 개념

Grid 레이아웃은 컨테이너와 아이템으로 구성됩니다:

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 200px;
  gap: 20px;
}
```

위 코드는 3개의 열과 2개의 행을 가진 그리드를 생성합니다.

## fr 단위 이해하기

`fr` 단위는 그리드 컨테이너의 사용 가능한 공간에 대한 비율을 나타냅니다:

```css
.container {
  grid-template-columns: 1fr 2fr 1fr;
}
```

이 예제에서는 가운데 열이 양쪽 열보다 두 배 넓습니다.

## Grid 영역 지정하기

Grid 아이템의 위치와 크기를 정확히 제어할 수 있습니다:

```css
.item {
  grid-column: 1 / 3; /* 1번 라인부터 3번 라인까지 */
  grid-row: 2 / 4; /* 2번 라인부터 4번 라인까지 */
}
```

또는 grid-area 속성을 사용할 수도 있습니다:

```css
.item {
  grid-area: 2 / 1 / 4 / 3; /* row-start / column-start / row-end / column-end */
}
```

## 자동 배치 알고리즘

Grid는 명시적으로 위치를 지정하지 않은 아이템을 자동으로 배치합니다:

```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: 20px;
}
```

`auto-fill`과 `minmax()`를 조합하면 반응형 그리드를 쉽게 만들 수 있습니다.

## 정렬하기

Grid 내에서 아이템을 정렬하는 다양한 방법이 있습니다:

```css
.container {
  justify-items: center; /* 가로축 정렬 */
  align-items: center; /* 세로축 정렬 */
}

.item {
  justify-self: end; /* 개별 아이템 가로축 정렬 */
  align-self: start; /* 개별 아이템 세로축 정렬 */
}
```

## Grid vs Flexbox

- Grid: 2차원 레이아웃(행과 열)에 적합
- Flexbox: 1차원 레이아웃(행 또는 열)에 적합

두 기술은 상호 보완적이며, 종종 함께 사용됩니다.

## 결론

CSS Grid는 현대 웹 디자인에 필수적인 도구입니다. 복잡한 레이아웃을 쉽게 구현할 수 있으며, 반응형 디자인에도 매우 유용합니다. 이 가이드를 통해 CSS Grid의 기본 개념부터 고급 기술까지 익히셨길 바랍니다.
